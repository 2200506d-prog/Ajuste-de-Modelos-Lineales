Análisis de Regresión Lineal

Analiza una tabla de datos de Excel o un conjunto de datos que ingreses, donde posteriormente realizará un ajuste de líneas rectas por medio de matrices y explicará este procedimiento que realiza para las personas que no comprenden del todo este código y sea de ayuda.

Cargar Excel

Selecciona un archivo .xlsx donde la Columna A sea X y la Columna B sea Y.

Entrada Manual

📖 Explicación del Procedimiento

El ajuste se realiza mediante la Ecuación Normal de Gauss:

(Aᵀ A)x = Aᵀ y
  1. Matriz A: Registra los valores X y una columna de 1s (término independiente).
  2. Matriz de Covarianza (AᵀA): Resume las sumatorias de los datos para el sistema.
  3. Inversa: Resuelve el sistema para hallar la pendiente y el intercepto.
